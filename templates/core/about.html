About MMT Budget Response Suite

The MMT Budget Response Suite is a real-time platform for tracking, fact-checking, and rebutting budget myths from a Modern Monetary Theory perspective.

Why This Platform?

Every budget day, the same economic myths are repeated: "taxpayer money," "maxed out credit card," "balancing the books." This platform provides tools to:

  • Track these myths in real-time with Budget Bingo
  • Crowdsource and AI-verify fact-checks with MMT analysis
  • Generate comprehensive rebuttals for activists, MPs, and media

How It Works

Budget Bingo

Generate a 5x5 bingo card with common budget myths. Mark them as you hear them during the Chancellor's speech. Compete with others for fastest completion!

Fact-Check Platform

Submit claims you hear during the budget for AI-powered fact-checking. Each response includes:

  • The Claim: Restated clearly
  • The Problem: Why it's misleading
  • The Reality: What's actually true
  • The Evidence: Citations and data
  • The MMT Perspective: How MMT reframes the issue

Comprehensive Rebuttal

AI-generated comprehensive analysis document including:

  • Executive Summary
  • Key Myths Debunked
  • Fiscal Analysis
  • Action Points for activists, MPs, and media

Technology

Built with Django, HTMX, and Tailwind CSS. Powered by Claude AI for intelligent fact-checking and rebuttal generation.
